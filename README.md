# -tetracomposibot-multi-agent-simulator
Multi-agent robot simulation in Python

## Description
Ce projet implémente une simulation d’une arène dans laquelle plusieurs robots interagissent en temps réel.  
Chaque robot possède un comportement spécifique (exploration, évitement, stratégie), permettant d’étudier les systèmes multi-agents.

## Fonctionnalités
- Simulation de robots dans une arène
- Déplacement autonome des agents
- Détection d’obstacles et d’autres robots
- Implémentation de comportements (Braitenberg, aléatoire, stratégique)
- Simulation en temps réel

## Structure du projet
- `tetracomposibot.py` : fichier principal
- `robot_*.py` : comportements des robots
- `config*.py` : paramètres de simulation

## Technologies
- Python
- PyGame
- Numba

## Objectif
Analyser les comportements émergents dans un système multi-agents.
